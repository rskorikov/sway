# Just a personal Sway configuration. No more, no less.
### Install Sway on Arch linux
Use command:
```sudo pacman -S sway swaybg foot polkit rofi``` on Arch linux

or just choose Sway while installing Arch linux with ```archinstall``` script.

Start Sway with ```sway``` command from TTY or use display manager.

Start terminal (foot is default) with ```Super + Return```

Create directory for default config file with:
```mkdir -p .config/sway```

Copy default config file to created directory:
```cp /etc/sway/config .config/sway/config```
and edit it to your liking/
